# Rock Paper Scissors 🎮

A simple **Rock-Paper-Scissors** game implemented in **C, C++, and Python**.

---

## 📂 Project Structure
```
C/       → C implementation
CPP/     → C++ implementation
Python/  → Python implementation
```

---

## ▶️ How to Run

### C
```bash
cd C
gcc rps.c -o rps
./rps
```

### C++
```bash
cd CPP
g++ rps.cpp -o rps
./rps
```

### Python
```bash
cd Python
python3 rps.py
```
